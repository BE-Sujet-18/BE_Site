TP 5 SCRATCH

🍄 Mario Bros : Le Défi des Goombas ! 🏃‍♂️

Dans ce TP Scratch, tu vas créer un mini-jeu inspiré de Mario !
Tu vas apprendre à faire sauter Mario, éviter les Goombas et gagner des points. 🎮


🌟 Objectif :

- Déplacer Mario avec les flèches du clavier
- Gérer les sauts avec effet de gravité
- Ajouter des ennemis Goombas qui défilent
- Compter les points à chaque évitement réussi
- Terminer le jeu en cas de collision


🧠 Notions de programmation que tu vas découvrir :

  • 🟢 Les événements : déclencher le jeu avec le drapeau vert
  • ⬆️ La gravité : faire sauter Mario et le faire redescendre
  • 🎯 Les capteurs : détecter les collisions avec les ennemis
  • 🔁 Les boucles : faire défiler les Goombas en boucle
  • 📦 Les variables : gérer un score qui augmente
  • 👯 Les clones : ajouter plusieurs Goombas dynamiques

💡 Défis supplémentaires :

- Ajoute plus de Goombas avec des vitesses différentes !
- Intègre un chronomètre pour tester ta survie ⏱️
- Change le décor pour explorer d'autres mondes 🎨

🏁 Résultat final :

Un mini-jeu Mario complet, drôle et personnalisable, avec sauts, ennemis et score.
Tu vas adorer voir Mario bondir au rythme du code ! 🍄✨
